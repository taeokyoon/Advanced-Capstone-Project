# 트레이 앱
프로그램창을 띄워두지 않고, 윈도우 오른쪽아래 알림영역 (시계옆 아이콘)에서
동작하는 프로그램을 트레이 앱이라고 합니다.

핵심은
1. 백그라운드 실행 중심 앱
2. 트레이 아이콘 우클릭 메뉴로 시작/중지/종료 같은 기능을 제공
3. 항상 켜져있지만 방해는 적은 형태에 적합
ex) 화면 자세 모니터링, 메신저, 클립보드 도구, 업데이트 감시 프로그램 같은 앱

# pystray
트레이 아이콘/메뉴를 만드는 UI 라이브러리

# 캘리브레이션 (calibration)
보통 "앱 로직"에서 하는 기준값 설정
ex) 바른 자세일 때 score를 저장해두고, 이후 편차를 비교하는 과정이 캘리브레이션

# 파이썬의 가상환경 venv ?
venv는 파이썬 패키지/인터프리터를 프로젝트별로 분리하는 "파이썬 전용 격리환경"

venv는 ubuntu로 볼 수 있는가? -> X
ubuntu는 리눅스 운영체제임.
비유하자면 ubuntu는 집(운영체제)라면 venv는 집 안의 방 하나(프로젝트 별 파이썬 환경)라고 볼 수 있다
즉, ubuntu 위에서도 venv를 만들 수 있고, 윈도우에서도 venv를 만들 수 있음

# Json과 Jsonl 차이
json은 javaScript Object Notation로, 데이터를 저장하고 전송하기위해 경량 텍스트 형식임. 키-값 쌍으로 구성되며, 사람과 기계가 읽기 쉬움

Jsonl은 json lines의 약자로, 각 줄에 하나의 json 객체가 있는 텍스트 파일의 형식임. 큰 데이터를 스트리밍하거나 로그파일로 처리할 때 유용함
